🧱 Техническая архитектура
Фреймворк: Да, используется React + Vite

Роутинг: Да, используется wouter

Рендеринг: Используем SPA (client-side rendering). SSR/SSG не требуется

Страницы:

/blog.tsx — страница списка блогов + страницы отдельных постов /blog/:slug

/case-studies.tsx — список кейсов + страницы кейсов /case-studies/:slug
